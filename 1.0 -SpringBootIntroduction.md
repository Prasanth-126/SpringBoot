1ï¸âƒ£5ï¸âƒ£ Interview-Ready Explanation
Q: What is Spring Boot?

Spring Boot is a Java-based framework built on top of Spring that simplifies application development by providing auto-configuration, embedded servers, and starter dependencies. It helps build production-ready applications quickly with minimal configuration.

Q: Why Spring Boot is popular?

It reduces development time, eliminates boilerplate configuration, supports microservices architecture, and integrates easily with databases and external APIs.

Q: Difference between Spring and Spring Boot?
Spring	Spring Boot
Manual configuration	Auto-configuration
External server required	Embedded server
XML heavy	Minimal configuration
Complex setup	Quick startup
