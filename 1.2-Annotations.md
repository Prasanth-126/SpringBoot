# 1Ô∏è‚É£ **@SpringBootApplication**

## ‚úÖ What it is

`@SpringBootApplication` is the **entry point annotation** that starts a Spring Boot application.

Internally it includes:

*   `@Configuration`
*   `@EnableAutoConfiguration`
*   `@ComponentScan`

## ‚úÖ Why we use it

To:

*   Start the Spring application
*   Auto-configure required beans
*   Scan packages and create components

## ‚úÖ Where we use it

On the **main class** of your Spring Boot project.

## ‚úÖ How to use (Real-time example)

```java
@SpringBootApplication
public class HospitalApp {
    public static void main(String[] args) {
        SpringApplication.run(HospitalApp.class, args);
    }
}
```

üí° This boots the entire Hospital Management application.

# 2Ô∏è‚É£ **@RestController**

## ‚úÖ What it is

Annotation that marks a class as a **REST API controller**.

Equivalent to:

    @Controller + @ResponseBody

## ‚úÖ Why we use it

To return JSON responses instead of HTML.

## ‚úÖ Where we use it

In the **controller layer** for REST APIs.

## ‚úÖ How to use (Real-time example)

```java
@RestController
@RequestMapping("/patients")
public class PatientController {

    @GetMapping("/{id}")
    public Patient getPatient(@PathVariable Long id) {
        return new Patient(id, "Ramesh", 29);
    }
}
```

# 3Ô∏è‚É£ **@Controller**

## ‚úÖ What it is

Marks a class as a **web MVC controller**.

## ‚úÖ Why we use it

For returning **HTML views** (Thymeleaf/JSP).

# 4Ô∏è‚É£ **@Service**

## ‚úÖ What it is

Annotation that marks a class as a **service**, meaning it holds business logic.

## ‚úÖ Why we use it

To separate:

*   Controller (API layer)
*   Service (Business logic)
*   Repository (Data access)

## ‚úÖ Where we use it

In **service/logic layer**.

## ‚úÖ How to use (Real-time example)

```java
@Service
public class PatientService {

    public String registerPatient(Patient p) {
        return "Patient registered: " + p.getName();
    }
}
```

# 5Ô∏è‚É£ **@Repository**

# 7Ô∏è‚É£ **@Autowired**

## ‚úÖ What it is

Injects a dependency into a class.

## ‚úÖ Why we use it

To avoid manually creating objects (Spring handles it).

## ‚úÖ Where we use it

Anywhere you need **dependency injection**.

## ‚ùå Field Injection (Not recommended)

```java
@Autowired
private PatientService service;
```

## ‚úî Constructor Injection (Best Practice)

```java
@Service
public class PatientController {

    private final PatientService service;

    public PatientController(PatientService service) {
        this.service = service;
    }
}
```

# 8Ô∏è‚É£ **@GetMapping, @PostMapping, @PutMapping, @DeleteMapping**

## üéØ What these are

These define **HTTP methods** in Spring Boot.

| Annotation       | Purpose       |
| ---------------- | ------------- |
| `@GetMapping`    | Retrieve data |
| `@PostMapping`   | Create data   |
| `@PutMapping`    | Update data   |
| `@DeleteMapping` | Delete data   |

## üî• Real-time Example

```java
@RestController
@RequestMapping("/patients")
public class PatientController {

    @PostMapping
    public Patient savePatient(@RequestBody Patient p) {
        return repo.save(p);
    }

    @GetMapping("/{id}")
    public Patient getPatient(@PathVariable Long id) {
        return repo.findById(id).orElse(null);
    }
}
```

# 1Ô∏è‚É£7Ô∏è‚É£ **@ControllerAdvice**

## ‚û§ What it is

### Real-time Example

```java

    }
}
```
